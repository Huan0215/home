# Admin System

This is a simple Spring Boot + MyBatis + MySQL admin system example with Thymeleaf.

Quick start:
1. Clone the repo
2. Update src/main/resources/application.properties with your MySQL username/password
3. Import src/main/resources/schema.sql and data.sql into MySQL (or let Spring run them if configured)
4. mvn clean package
5. java -jar target/admin-system-0.0.1-SNAPSHOT.jar

Default login: admin / password
